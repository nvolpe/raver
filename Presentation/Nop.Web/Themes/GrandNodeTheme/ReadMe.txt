 Added possbility to change colors:
 available colors are set at GrandNodeTheme\Content\css\colors
 
Predefined colors:
	Brackwhite
	Blue
	Bluegray
	Gray
	Green
	Lightblue
	Orange
	Purple
	Red

customValuess - in that file you may set your own colors. File available as a css and scss file. 

To use predefined color, open \Themes\GrandNodeTheme\Views\Shared\Head.cshtml file, comment line #27 and uncomment desired color.
